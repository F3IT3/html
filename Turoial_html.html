Tutorial HTML

Para títulos usa <h1> ou até o <h6> fechando a tag

Para paragrafo usa <p> e </p>

Para Linha horizontal <hr>

Para Quebra de linha usa <br>

Para Comentário <!-- -->

Para usar Emojis &#x"código do emoji" 🔥 😄

Para Negrito use a tag <strong>

Para Itálico use a tag <em>

Para Marcar um texto use a tag <mark>

Para texto Grande use <big>

Para texto Pequeno use <small>

Para texto "deletado" use <del>

Para texto sublinhado use <ins>

Para elevar use <sup>, x2+2=6

Para descer use <sub>, H2O

Para os caracteres terem mesmo espaço use <code>

Para citações semânticas <q>

Para citações completas use <blockquote>

Para abreviar use <abbr>

Para entrar no mundo invertido <bdo> , Este é o poder da Sakanade

Para usar Lista Ordenada <ol>

Para usar Lista Desordenada <ul>

Para usar Item de Lista <li>

Para usar Lista de Definição <dl>

Para usar um Link <a href="">

Para usar uma Imagem <img src="" alt="">

Para usar uma Imagem Dinâmica <picture>
